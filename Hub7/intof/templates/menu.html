There are only 3 API calls using HTTP POST :
(All paths shown below are relative paths)

1) To onboard a new device:

Make a POST request to /onboard/device
Paylaod:
{
"device_id" : "ABC1234",
"relesn_list" : ["POWER1", "POWER2", "SENSOR"]
}

2) To enable/disable a device:

Make a POST request to /update/device
Payload:
{
"device_id" : "ABC1234",
"enabled" : true
}

3) To update a relsen object:

Make a POST request to /update/relsen
Payload:
{
"device_id" : "ABC1234",
"relsen_id" : "POWER3",
"relsen_name" : "Bed room fan",
"relsen_type" : "Fan",
"room_name" : "Mother\'s bed room",
"room_type" : "Bed room",
"group_name" : "Ground floor",
"schedule" : [[11.45,12.0], [13.0,14.30]]}"
}
